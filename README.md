# car-sales-sql-analysis

## Project Overview
This project analyzes a car sales dataset using MySQL. The goal is to generate business insights by joining multiple relational tables and writing SQL queries.

## Dataset
- Sales
- Customers
- Models
- Dealership
- Geography
- Leads
- Test Drive

## Tools Used
- MySQL Workbench
- SQL
- GitHub

## SQL Concepts Used
- Joins
- CTE
- Aggregate Functions
- Window Functions
- GROUP BY
- HAVING
- CASE WHEN
- ORDER BY

## Business Questions Solved

1. Find total sales by dealership.

2. Identify the age group that purchases the most cars.

3. Find the top 3 cities with the highest sales.

4. Find the second highest-selling car model using `DENSE_RANK()`.

5. Count the number of male and female customers who purchased cars.

6. Identify the month that generated the highest revenue.
